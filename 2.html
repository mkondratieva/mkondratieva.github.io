Срок сдачи двух задач на классы (простой и сложный) 1 октября. Каждая задача оценивается одним баллом. На сроке сдачи задач 8.10-29.10 снимается (в сумме) 0,5 балла (т.е. с каждой по 0,25). На более поздних сроках -- 1(с каждой по 0,5). Решения можно отправлять на почту для консультации, сдавать в классе на учебных ПК.

Программа должна быть разделена по трем файлам. Нельзя использовать библиотеку STL. Компилировать с использованием ключей

g++ -fno-elide-constructors -std=c++11

Файлы заголовков включаются в cpp-файлы с помощью директивы #include, их не надо компилировать.

Задача номер 3 динамические структуры данных, (C++).

Срок сдачи -- до 19 ноября включительно. Оценивается из 4 баллов. Пропуск срока влечет снижение баллов (минус 1 балл). Задача выдается после того, как сделаны две первые.

На некоторых семинарских занятиях можно получить 0.25 балла, выполнив задачу в классе (нельзя забирать на дом). Задача может, например, включать создание шаблона стека и требовать некоторых несложных дополнительных действий. Другим примером является рекурсивный поиск одновременно двух порядковых статистик и развертка рекурсии с помощью очереди. Программу можно найти в этой папке.. 5.11 было задание реализовать волновой алгоритм на неориентированном графе, в той же папке файл wave.cpp, первое использование STL. Детали описаны в самом файле. 12.11 было предложено перегрузить операцию сложения для списков, используя готовый шаблон, взятый из лекций. Под сложением списков l1+l2 понималось добавление к списку l1 после текущего всех элементов l2 (с сохранением их порядка). С решением можно ознакомиться здесь.. Все остальные файлы проекта находятся в этой же папке. Шаблон списка с фиктивным элементом можно использовать в задаче 3, срок сдачи которой 19 ноября. Для получения максимальной оценки за это задание (4 балла) нужно не только принести в класс готовое решение, но и уметь добавить некоторую несложную возможность работы с классом.

Задача номер 4 вычислительная геометрия (C++, Python).

Срок сдачи -- до 10 декабря включительно. Оценивается из 2 баллов (отдельно С++ и Python). Пропуск срока влечет снижение баллов (минус 0.5 балла).
