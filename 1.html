Работа на ЭВМ, 1 курс, механики, 2024-2025г.

В 3 семестре программы будем писать на С++ и Python. Но все, что касается С, будет включено в экзамен (зимой). Летом будет полезно максимально самостоятельно написать полную (любую) ссылочную реализацию списка на С и С++, включая 2 структуры (узел списка и сам список) и базовые операции работы со списком (методы узла и списка). Задачу, использующую список, можно взять такую.

Дано изображение в формате bmp (например, изображение с палитрой, 8 бит на пиксел, RGB-модель, см. справку и, например, это видео, на сайте лектора можно найти и более позднюю его версию). Построить список черных точек изображения (предполагается, что черный цвет в палитре есть). Отсортировать список черных точек по возрастанию расстояния до левой нижней вершины изображения. За расстояние между пикселями взять максимум модуля разности их координат, при равенстве расстояния -- сравнивать модули разностей по каждой из осей. Следующую процедуру выполнять, пока изменяется изображение. Перекрасить в черный цвет середину отрезка, соединяющего две соседние точки отсортированного списка. Если середина этого отрезка не принадлежала списку черных точек, добавить ее в список, и эту новую точку далее в текущем обходе списка не рассматривать. Текущий обход списка заканчивается после обработки последней пары соседних пикселей. Рассматриваются все соседние точки, т.е. если список состоит из пикселей x_0,...x_n-1, то берем середины отрезков [x_0, x_1],...,[x_n-2, x_n-1].

Ссылка на видео об установке необходимого ПО (Win) (В.М.Староверов). Файлы для скачивания (компилятор gcc + FAR). Следует выбрать свой вариант операционной системы (32 или 64-битная), скачать и запустить файл. Возможно, ваш антивирус будет выдавать предупреждение, которые придется игнорировать.

Пакетные файлы (жесткие ключи) wgcc, wg++ для Win (.bat) и Linux (без расширения). Для установки (Win) после распаковки положите файлы в директорию, где находится gcc.exe. Инструкцию по установки для Linux можно найти здесь (строки 62-77). Для проверки утечек памяти (Win, Си) файл malloc.c необходимо вставить в ваш проект. Аналогично для проверки утечек памяти (Win, С++) файл malloc.cpp необходимо вставить в ваш проект.

Компилируем только из командной строки, среду использовать нельзя! Задачи принимаются по порядку и только в виде проекта.
