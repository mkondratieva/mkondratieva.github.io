2.09. Понятие индуктивной функции. Блок-схема для вычисления значения индуктивной функции и пример программы, вычисляющей количество элементов файла.

12.09. Типы данных в языке Си. (целые, вещественные).

23.09. Указатели. Пример функции с аргументом, который является указателем на результат.

Начиная с 4 задачи программы принимаются в виде проекта. У вас должно быть 3 файла в проекте: заголовочный, файл с реализацией, файл с тестом на работу функции. Также к проекту необходимо приложить Makefile для сборки проекта. Команда, по которой проект собирается, зависит от версии компилятора, обычно это make.

31.10. Массивы; переворот, сдвиг, уплотнение к началу (рекурсивные реализации).

7.11. Динамические массивы. Функции malloc, realloc. Для проверки утечек и ошибок сегментации желательно установить Ubuntu на свой ноутбук (ПК). В классе необходимо проверить свою программу по такой схеме: valgrind ./a.out

14.11. Массивы. Сепарирование, проход пузырьком. Рекурсивные варианты.

21.11. Массивы. Сортировка пузырьком и простыми вставками. Рекурсивные варианты.

С компилятором работаем только из командной строки, среду использовать нельзя!
