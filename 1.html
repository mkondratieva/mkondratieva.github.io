Тема 1, последовательности. Список задач.

Список заданий контрольной работы можно посмотреть здесь. Решения этих задач можно найти в этой папке.

В программе должна присутствовать функция, вычисляющая нужную характеристику последовательности чисел из файла. В ней не должно быть печати. Ошибки (пустой, плохой файл) должны быть обработаны. В командную строку можно попасть, открыв в домашнем каталоге папку, в которой находится файл с программой (в примере это 1.c), в меню, которое появится после нажатия правой кнопки мыши, выбрать "действие"-> "открыть терминал в этой папке". Компилируем программу из командной строки, например,

gcc 1.c

Тогда результатом должен быть файл a.out. Запустить на выполнение можно командой

./a.out
