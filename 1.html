Чтобы в Linux скачать с github публичный репозиторий (пароль не требуется): git clone https://github.com/mkondratieva/1kurs.git

Ссылка на видео об установке необходимого ПО (Win) (В.М.Староверов). Файлы для скачивания (компилятор gcc + FAR). Следует выбрать свой вариант операционной системы (32 или 64-битная), скачать и запустить файл. Возможно, ваш антивирус будет выдавать предупреждения, которые придется игнорировать.

Пакетные файлы (жесткие ключи) wgcc, wg++ для Win (.bat) и Linux (без расширения). Для установки (Win) после распаковки положите файлы в директорию, где находится gcc.exe. Инструкцию по установки для Linux можно найти здесь (строки 62-77). Для проверки утечек памяти (Win, Си) файл malloc.c необходимо вставить в ваш проект.

Компилируем только из командной строки, среду использовать нельзя!

Краткая инструкция по работе в Linux.

Если дома вы работаете не в Linux, по возможности установите вторую ОС. Как можно установить две операционные системы, советы: Win, Mac . По последней ссылке мой опыт работы на Мас и инструкции по установке гостевой Ubuntu в Virtual Box (для любой ОС).
