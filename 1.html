Задание 1. Двумерные массивы известного размера. Срок сдачи -- 25 февраля включительно. Список задач 1.

В программе должна присутствовать функция, отводящая память под (не одномерный!) двумерный массив и заполняющая его числами из файла. Ошибки обработать. Количество чисел в файле должно соответствовать заявленным размерам (иначе -- выдача сообщения). При обработке ошибок можно не отслеживать количество чисел в строке, а только их общее количество в файле. Файл открываем один раз. Вся захваченная память должна корректно освобождаться (без утечек). Преобразование массива (удаление строк или столбцов) должно быть в отдельной функции. Программу разделить по 3 файлам: заголовочный, с функцией main и реализация всего остального. Печать можно делать на экран. Вариант без обработки ошибок. Еще один пример отводит память под матрицу единым куском.

11.02 в классе осваивали побитовые операции. Смоделируем множество целых чисел в диапазоне от 0 до 31 следующим образом. Пусть set - целое число, отождествим его с таким множеством: число 0<=i<=31 принадлежит set, если i-й бит set равен 1. Нужно было реализовать распечатку элементов множества, добавление(удаление) элемента, пересечение, объединение, дополнение и симметрическую разность. Решение.

18.02 нужно было распечатать двоичный порядок вещественного числа х типа float. Старший бит -- это знак числа, под порядок отведено 8 бит, предшествующих знаку. |x|=x0*2^n, 1<=x0<2 -- в таком выражении n -- двоичный порядок х, x0 -- нормализованная двоичная мантисса. Порядок хранится в битах с номерами 23-30, мантисса -- с номерами от 0 до 22 (включительно). Побитовые операции недопустимы с вещественным типом, поэтому для доступа к машинному представлению таких чисел необходимо преобразование адреса. Частичное решение.

Задание 2. Работа со строками. Срок сдачи -- 10 марта включительно. Список задач 2.

Для знакомства со строками рекомендуется прочитать следующий документ и посмотреть примеры программ 1, 2, 3, 4, и 5. В задачах требуется, вызвав в цикле fgets, просмотреть все строки файла, выделить слова, удовлетворяющие некоторому условию, по заданному критерию выбрать из них одно и распечатать его. Слова можно выделять вручную (используя strchr), с помощью функции sscaf или применяя strtok. Строки в файле имеют известное ограничение на размер, поэтому память можно выделять статически (в этом случае не будет ее утечек).

03.03 предлагалось найти количество слов заданной длины и распечатать все такие слова, каждое в отдельной строке. Решение двумя способами.

04.03 предлагалось записать в обратном порядке символы внутри каждого слова строки, разделители при этом не должны измениться. Решение тремя способами.

10.03 То же задание, но под словами понимать участок строки внутри двойных кавычек. Решение двумя способами.
